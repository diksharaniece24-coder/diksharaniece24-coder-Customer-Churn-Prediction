# Customer Churn Prediction

A machine learning project to predict whether a telecom customer is likely to churn based on customer data.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Project Files

- `Scripts/` - Project source files
- `icons/` - Icons used in the application
- `output/` - Saved outputs and model files
- `data.csv` - Dataset

## How to Run

```bash
pip install -r requirements.txt
python Scripts/app.py
```
